# Demo Sample

CSPP demo sample instance for GitHub demos. Store names, identifiers, addresses, coordinates, and demand rows are generated data. The warehouse is centered near a demo depot area, and stores are randomly placed within a 100 km radius. The demand profile only preserves broad scale and sparsity characteristics needed to exercise the web app.

- Seed: `20260424`
- Stores: `48`
- Demand rows: `1488`
- Files: `instance_payload.json`, `assignments.json`, `stores.geojson`

The included cluster IDs are demo angular sectors around the demo warehouse. The public instance payload does not include a clustering method field.
